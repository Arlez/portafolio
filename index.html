Diseño Web

El diseño web implica conocer cómo se deben utilizar cada uno de los elementos permitidos en el HTML, es decir, hacer un uso correcto de este lenguaje dentro de los estándares establecidos por la W3C y en lo referente a la web semántica.

Front End

El desarrollo Front End una especialidad para la creación de páginas web, que trabaja la interfaz mediante los lenguajes HTML, CSS y Javascript, con esto permite que el usuario pueda interactuar con nuestra web.

Responsive

Una página web con diseño responsivo es una técnica de diseño web que brinda a un sitio la capacidad de adaptarse a diferentes tamaños de pantallas a través de la reorganización del contenido e imágenes y simplificación del menú.
